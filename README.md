# sipenurut

Si Penurut (Sistem Informasi Pencari Guru Private )

![Repo_List](assets/users/images/home/SiPenurut.png)

## Teknologi

- PHP
- Codeigniter
- MySQL
- Bootstrap

## Instalasi

Download dan ekstrak ke htdocs,

Buat database dengan nama "sipenurut" dan import file database.sql

```bash
sipenurut
```
